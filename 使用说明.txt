抖音视频工具 - 小白使用说明

最简单用法：
1. 双击：打开抖音操作页面-无黑窗.vbs
2. 浏览器会自动打开一个页面
3. 把抖音分享内容粘贴进去
4. 点击：解析信息 / 获取下载链接 / 下载视频
5. 下载时页面会显示进度条和百分比
6. 需要视频语音转文字时，填写 API Key 后点击：提取文案
7. 文案提取完成后，可以点击：复制文案 / 打开文案
8. 下载的视频和提取的文案会保存到 downloads 文件夹
9. 已下载列表默认每页 5 个，可以切换每页数量并翻页
10. 已下载列表里有序号、动作、文件名和打开按钮

批量任务：
1. 在“批量导入链接”里一次粘贴 100 / 500 / 1000 条抖音分享内容
2. 系统会自动识别抖音链接、去重，并写入 SQLite 队列
3. 状态包括：等待、下载中、提取中、完成、失败
4. 可以选择并发 1 / 3 / 5，默认同时下载 3 个
5. 开启“下载后提取文案”时，会使用已保存的 DashScope API Key 自动提取文案
6. 开启“生成 AI 分析”时，会为文案生成爆款钩子、情绪点、痛点、行动号召和标签
7. 队列数据保存在 .data\tasks.sqlite
8. 支持导出 CSV / XLSX
9. 当前版本先支持作品链接批量下载；账号、合集、评论采集、点赞收藏统计已预留入口，后续接入 TikTokDownloader 后启用

后台说明：
1. 页面打开时，后台会自动隐藏运行
2. 页面关闭后，后台会自动退出
3. 如果页面异常卡住，再双击：关闭抖音操作页面后台.bat

如果页面打不开：

2. 再双击：打开抖音操作页面-无黑窗.vbs

是否需要 API：
1. 解析链接：不需要 API
2. 获取下载链接：不需要 API
3. 下载公开视频：不需要 API
4. 自动提取文案/语音转文字：需要阿里云百炼 DashScope API Key

如果后面要加“自动提取文案”：
当前已经接入阿里云百炼 DashScope。
在页面里输入 API Key 并保存即可。
