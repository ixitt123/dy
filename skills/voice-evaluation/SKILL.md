# 声音评估

## 测试维度

1. 日常对话：自然度和亲和力。
2. 招生话术：可信度、重点表达和行动引导。
3. 情绪表达：情绪变化和表现力。
4. 长文本：稳定性、断句和耐听度。
5. 短视频口播：节奏、开场吸引力和口语感。

## 评分规则

- 星级：1 至 5 星。
- 分数：0 至 100 分。
- 可附加简短评价。
- 每段测试保存独立评分；同一段测试再次评分时更新原记录。

## 执行要求

- 测试脚本从 `prompts/voice_test_script.md` 读取。
- 五段测试通过现有 TTS 队列生成。
- Provider 不支持的参数继续按 TTS Adapter 规则忽略并记录。
- 测试失败不得影响声音资产本身。
