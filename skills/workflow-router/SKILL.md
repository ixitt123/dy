# Workflow Router

Route:

- `chart`: verify data → define axes and labels → create chart specification
- `infographic`: extract facts → build hierarchy → define visual blocks
- `ai_image`: validate necessity → prepare positive and negative prompts
- `stock_video`: define search terms → license requirement → crop and trim instructions
- `text_motion`: define text hierarchy → entrance, emphasis and exit timing
- `screenshot`: identify source page → capture state → redact sensitive data
- `ui_demo`: define interaction path → capture steps → annotate focus
- `real_photo`: define required subject, consent, location and framing
- `document`: identify document source → crop, highlight and privacy checks
- `mixed`: define layer order and why composition is necessary

Do not route to a Provider in phase one.
