# Scene Composer

## 构图顺序

1. 确定镜头唯一目的。
2. 选择主体与环境，避免抽象空话。
3. 安排前景、中景、背景，建立空间纵深。
4. 选择光源、时间、色调和真实材质。
5. 规定主体动作与可见证据。
6. 为平台比例保留字幕安全区。

## Composition 写法

必须描述：

- 地点与时间
- 主体和关键物品
- 前景、中景、背景关系
- 光线与色调
- 画面重点和字幕安全区

禁止只写“一个学生”“一个学校”“高级画面”等空泛描述。
