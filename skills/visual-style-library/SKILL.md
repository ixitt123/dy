# Visual Style Library

## education-premium

- 适合：教育机构、校区宣传、课程介绍。
- 关键词：干净、高级、可信、明亮、克制、专业。
- 色调：中性白、深绿、少量暖色点缀。
- 禁止：廉价招生海报风、土味红黄配色、PPT 模板感。

## parent-anxiety

- 适合：家长焦虑、成绩问题、升学风险。
- 关键词：压迫感、现实感、生活化、情绪递进。
- 色调：低饱和冷暖对比，保留真实室内光。
- 禁止：恐吓式营销、虚假承诺、过度煽动。

## knowledge-blogger

- 适合：抖音知识口播、学习方法。
- 关键词：快节奏、字幕强、信息密度高、强观点。
- 构图：人物口播、信息图、细节插入和文字卡交替。
- 禁止：无意义空镜、重复人物镜头、拖节奏。

## apple-keynote

- 适合：高端课程、品牌介绍。
- 关键词：极简、留白、精致、慢节奏、重点突出。
- 色调：黑、白、灰与单一品牌强调色。
- 禁止：杂乱背景、低级特效、过多装饰。

## documentary

- 适合：真实案例、校区故事、学生成长。
- 关键词：真实、生活化、纪实、自然光、情绪克制。
- 构图：观察式跟拍、环境建立、人物细节。
- 禁止：过度摆拍、假大空、虚假仪式感。

## cinematic

- 适合：品牌片、活动预热视频。
- 关键词：电影感、光影、纵深、空间感、情绪氛围。
- 构图：前中后景、运动镜头与有动机的光线。
- 禁止：AI 图库拼贴、镜头重复、只追求空洞氛围。

## clean-commercial

- 适合：课程卖点、产品化服务、活动信息。
- 关键词：清晰、直接、现代、产品聚焦、可信。
- 禁止：过度包装、复杂背景、信息堆叠。

## warm-campus

- 适合：校区生活、师生互动、成长记录。
- 关键词：温暖、自然、生活化、明亮、有人情味。
- 禁止：假笑摆拍、过曝滤镜、千篇一律人物特写。
