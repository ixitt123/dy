# Subtitle Timeline

## 规则

- 时间轴从 0 秒开始，连续且不得重叠。
- 每条字幕对应当前镜头的主要信息。
- 单条字幕优先控制在 8-18 个汉字，长句按语义切分。
- highlight 只保留 1-3 个关键词。
- 字幕使用干净中文无衬线字体，避免描边堆叠和多色跳字。
- 竖屏字幕避开底部平台操作区和人物嘴部。
- 未提供 TTS 时长时，依据口播字数和节奏估算。
- 提供 TTS 时长时，镜头总时长应接近音频时长。
