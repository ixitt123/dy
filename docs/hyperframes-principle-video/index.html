Render Pipeline

浏览器逐帧捕获，最后编码成 MP4

1. HTML/CSS 布局先定稿，保证每一帧都可预览。
2. Timeline 确定性动画让 seek、检查和渲染结果一致。
3. Capture Headless Chrome 抓帧，再交给 FFmpeg 输出。
source -> timeline -> frames -> mp4
