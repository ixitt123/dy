## 逐帧结构结论

实际画面不是复杂实拍剪辑，而是可模板化的横版知识卡：

1. 0.0-1.1s：暗橄榄背景和中央黑色光斑铺底，主标题第一行淡入。
2. 1.1-2.5s：绿色关键词“必备能力”放大入场，左上粒子散开，中心圆环出现。
3. 2.5-3.8s：标题保持，粒子逐步消失，顶部小标题准备常驻。
4. 3.8-4.2s：短暂空场，保留暗色光斑和顶部小标题。
5. 4.2-7.2s：第 1 张能力卡入场，金色主标题、橙色副标题、细说明文字、横线和右侧光点。
6. 7.2-10.4s：第 2 张能力卡同构入场，文字内容替换，光点微脉冲。
7. 10.4-13.8s：第 3 张能力卡同构入场。
8. 13.8-15.1s：文字退出，背景暗下，结束。

## 可复刻模板规则

- 背景：暗橄榄径向渐变 + 中央黑色光斑 + 轻微颗粒。
- 主色：金色标题、绿色关键词、橙色副标题、浅色正文。
- 动效：文字上移淡入、关键词放大、粒子散入散出、圆环缩放、卡片切换淡出淡入。
- 结构：标题页 + 空场过渡 + 三张能力卡 + 暗场收尾。
- 适用：职场管理、领导力、认知提升、课程方法论、三点式知识讲解。
