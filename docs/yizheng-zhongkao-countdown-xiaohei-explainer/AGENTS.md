# Yizheng Zhongkao Countdown Xiaohei Explainer

This folder is a test run of the `xiaohei-explainer-video` workflow.

Source files:

- `director-script.md` is the Director System package.
- `storyboard.json` is the machine-readable storyboard and VFX contract.
- `DESIGN.md` defines the visual identity for HyperFrames.
- `index.html` is the HyperFrames source composition.

Local preview logs, thumbnails, and rendered MP4 files are generated artifacts and should not be committed.
