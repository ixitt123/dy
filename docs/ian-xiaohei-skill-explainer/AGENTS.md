# Ian Xiaohei Skill Explainer

This folder is a self-contained HyperFrames explainer video project.

Source files:

- `director-script.md` is the Director System first-stage output.
- `storyboard.json` is the machine-readable shot plan derived from the director script.
- `DESIGN.md` is the visual identity gate for the composition.
- `index.html` is the HyperFrames source composition.

Generated media, renders, logs, and downloaded assets should stay local and must not be committed.
