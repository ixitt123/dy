# TTS 情绪与风格模板

将用户选择的情绪转换为简短、明确的朗读要求：

- 自然：自然、清晰、节奏平稳。
- 兴奋：更有活力，语气上扬，保持清晰。
- 严肃：稳重克制，重点词清楚。
- 亲切：温和、有陪伴感，避免生硬。
- 焦急：节奏略快、有紧迫感，但不要含混。
- 震惊：突出意外感和关键词。
- 温柔：柔和舒缓，停顿自然。
- 痞里带刺：口语化、有观点、略带讽刺，不使用攻击性表达。
- 家长劝告：像有经验的家长或老师耐心劝说。
- 招生转化：可信、清楚、有行动引导，避免夸张承诺。
- 自定义情绪描述：优先采用用户填写的描述。

Provider 支持原生情绪或 instruction 时传入对应参数。
不支持时合并到 `stylePrompt`；仍不支持则忽略，并在 `metadata_json` 记录。
