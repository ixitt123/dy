# Material Strategy

Decide the information function before the material:

- proof or trend → chart / infographic
- process or method → text_motion / infographic / real_photo
- software action → screenshot / ui_demo
- official evidence → document
- enrollment environment → real_photo: campus_photo / student_scene
- family anxiety → real_photo: family_scene / report_card / study_room
- visual atmosphere → stock_video
- specific unavailable concept → ai_image

`ai_image` is not the default. Every decision requires a concrete reason.
